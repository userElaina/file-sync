# file-sync
自用的文件同步。未经过充分测试，建议打开生成的shell脚本看一下。
***@***:~/log# ls -R from
```
from:
bigfile  exist  existe  nexist  nexiste  only_time.txt

from/bigfile:
'bingo - 副本 (2).txt'  'bingo - 副本 (3).txt'  'bingo - 副本.txt'   bingo.txt

from/exist:
edit.txt  nexist.txt  same.txt

from/existe:

from/nexist:
'新建 文本文档.txt'

from/nexiste:
```
***@***:~/log# ls -R to
```
to:
exist  existe  more.txt  only_time.txt

to/exist:
edit.txt  same.txt

to/existe:
```
